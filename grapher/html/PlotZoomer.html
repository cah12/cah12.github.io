PlotZoomer

Detailed Description

PlotZoomer provides stacked zooming for a plot widget.

PlotZoomer selects rectangles from user inputs ( mouse or keyboard ) translates them into plot coordinates and adjusts the axes to them. The selection is supported by a rubber band and optionally by displaying the coordinates of the current mouse position.

Zooming can be repeated as often as possible, limited only by maxStackDepth() or minZoomSize(). Each rectangle is pushed on a stack.

The default setting how to select rectangles is a PickerDragRectMachine with the following bindings:

  • Enum.MousePatternCode.MouseSelect1
    The first point of the zoom rectangle is selected by a mouse press, the second point from the position, where the mouse is released.
  • Enum.KeyPatternCode.KeySelect1
    The first key press selects the first, the second key press selects the second point.
  • Enum.KeyPatternCode.KeyAbort
    Discard the selection in the state, where the first point is selected.

To traverse the zoom stack the following bindings are used:

  • Enum.MousePatternCode.MouseSelect3, Enum.KeyPatternCode.KeyUndo
    Zoom out one position on the zoom stack
  • Enum.MousePatternCode.MouseSelect6, Enum.KeyPatternCode.KeyRedo
    Zoom in one position on the zoom stack
  • Enum.MousePatternCode.MouseSelect2, Enum.KeyPatternCode.KeyHome
    Zoom to the zoom base

The setKeyPattern() and setMousePattern() functions can be used to configure the zoomer actions. The following example shows, how to configure the 'I' and 'O' keys for zooming in and out one position on the zoom stack. The "Home" key is used to "unzoom" the plot.

zoomer = new PlotZoomer( plot );

zoomer.setKeyPattern( Enum.KeyPatternCode.KeyRedo, KeyI, Static.Key_Shift );

zoomer.setKeyPattern( Enum.KeyPatternCode.KeyUndo, KeyO, Static.Key_Shift );

zoomer.setKeyPattern( Enum.KeyPatternCode.KeyHome, KeyHome );


PlotZoomer is tailored for plots with one x and y axis.
