PlotPicker

Detailed Description

PlotPicker provides selections on a plot canvas.

PlotPicker is a Picker tailored for selections on a plot canvas. It is set to a x-Axis and y-Axis and translates all pixel coordinates into this coordinate system.

Constructor Documentation

PlotPicker (canvas)

Create a plot picker.

The picker is set to those x- and y-axis of the plot that are enabled. If both or no x-axis are enabled, the picker is set to xBottom. If both or no y-axis are enabled, it is set to yLeft.
