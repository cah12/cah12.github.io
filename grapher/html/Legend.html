Legend

Detailed Description

The legend widget.

The QwtLegend widget is a tabular arrangement of legend items. Legend items might be any type of widget, but in general they will be a QwtLegendLabel.

setLegendDiv (div)

Sets the html div element that will contain all html elements that comprise the legend. The default is $("#legendDiv"). Applications that change the default must provide significant layout management.

  • Parameters

div

html div element

Usage:

...

var legend = new Legend();

legend.setLegendDiv ($("#legendDiv1")); //Must be called before insertLegend()

plot.insertLegend(legend);

...
